## 2. 모바일 한글 줄바꿈 최적화
- 문제: iPhone 13에서 한글 텍스트 가독성 문제
- 해결: 이미지 대신 CSS와 HTML 조합 사용
- 장점: SEO 향상, 접근성 개선, 반응형 지원
- 추가된 CSS 클래스 (src/app/globals.css):
  * korean-text: 한글 텍스트 최적화
  * mobile-title: 모바일 제목 최적화
  * section-title: 섹션 제목 공통 스타일
- 적용 위치: 여러 섹션의 제목과 텍스트를 의미 단위로 자연스럽게 줄바꿈

## 3. 스크롤배너 성능 최적화
